About This Project

The Career Guidance System is a web-based tool that helps students and professionals explore career options that align with their interests, skills, and aspirations. It is built entirely with HTML, CSS, and vanilla JavaScript — no backend or framework required.

The quiz uses a simple category-scoring algorithm: each answer maps to one or more career categories (Technology, Analytical, Creative, Social). After all questions are answered, each career is scored based on how closely its categories match those chosen in your answers. The top three matches are displayed as your personalised recommendations.

Features

  • 6-question personalised career assessment quiz
  • 8 career profiles with salary, growth, skills & resources
  • Match scoring visualised as a percentage bar
  • Searchable and filterable careers browser
  • Detailed career modal with curated learning links
  • Fully responsive design for desktop and mobile

Tech Stack

  • HTML5 & Semantic markup
  • CSS3 (custom properties, grid, flexbox, animations)
  • Vanilla JavaScript (ES6+, Fetch API)
  • JSON data store (no database needed)
